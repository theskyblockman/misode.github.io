# Generators

## Loot Table (1.15+, at [here](https://misode.github.io/loot-table/))

You can create a way to add custom loot and anothers things with the loot table, this generator can be used to create it

## Predicate (1.15+, at [here](https://misode.github.io/predicate/))

You can create some verification with this like verify if a person sneak or sprint and can be used like this:
```execute as @e[predicate=mysuperdatapack:mysuperpredicate] run ...```

## Advencement (1.15+, at [here](https://misode.github.io/advancement/))

You can create a advencement with some trigger like location (xyz, biome and many others)

## Dimension (1.16.4+, at [here](https://misode.github.io/dimension/))

You can create a dimension from this generator with it, you can define: the Dimension Type (TO MANY THINGS), Generator (default, superflat and debug world (see more one the debug world [here](https://minecraft.gamepedia.com/Debug_mode))), Dimension Seed (TO MANY THINGS) and generator settings is some settings

## Dimension Type (1.16.4+, at [here](https://misode.github.io/dimension-type/))

It is the same thing of a part of dimension generator but you with the Dimension Type setting at Custom
